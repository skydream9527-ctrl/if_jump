# 吃了么 - 动画与特效规范

## 一、动画设计原则

### 1.1 设计理念
- **生动有趣**：动画要有活力，符合美食主题
- **反馈明确**：玩家操作后要有清晰的视觉反馈
- **流畅自然**：动画过渡平滑，不突兀
- **性能优先**：平衡效果与性能，确保60fps

### 1.2 动画风格
- **弹性动画**：使用spring物理效果，有Q弹感
- **夸张表现**：适度夸张，增强趣味性
- **节奏明快**：符合休闲游戏轻松氛围
- **主题化**：与章节美食主题相呼应

### 1.3 性能标准
- **目标帧率**：60fps
- **动画预算**：同屏不超过10个复杂动画
- **内存限制**：动画纹理不超过20MB
- **加载策略**：按需加载，预加载常用动画

## 二、角色动画

### 2.1 阿饱角色动画状态机

```
[待机] → [蓄力] → [跳跃] → [空中] → [着陆]
  ↑                                    ↓
  └────────────────←←←←←←←←←←←←←←←←┘
         [失败] → [复活]
```

### 2.2 详细动画列表

#### 待机（Idle）
```
时长：循环
动画内容：
- 呼吸动画：scaleY 1.0 → 1.02 → 1.0
- 频率：2秒/次
- 附加：偶尔眨眼、左右张望

示例代码：
{
  "name": "idle",
  "duration": 2.0,
  "loop": true,
  "keyframes": [
    { "time": 0, "scaleY": 1.0 },
    { "time": 1.0, "scaleY": 1.02, "easing": "easeInOut" },
    { "time": 2.0, "scaleY": 1.0 }
  ]
}
```

#### 蓄力（Charge）
```
触发：按住屏幕
时长：最长1.5秒
动画内容：
- 身体下蹲：scaleY 1.0 → 0.8
- 蓄力特效：身体周围出现光圈
- 抖动效果：高频微抖动，随蓄力增强

阶段：
- 0-30%：轻微下蹲，光圈白色
- 30-70%：明显下蹲，光圈黄色
- 70-100%：最大下蹲，光圈橙色，强烈抖动

示例代码：
{
  "name": "charge",
  "duration": 1.5,
  "loop": true,
  "keyframes": [
    { "time": 0, "scaleY": 1.0, "glowColor": "#FFFFFF" },
    { "time": 0.45, "scaleY": 0.85, "glowColor": "#FFD700" },
    { "time": 1.05, "scaleY": 0.8, "glowColor": "#FFA500" },
    { "time": 1.5, "scaleY": 0.8, "glowColor": "#FF6B6B" }
  ]
}
```

#### 跳跃（Jump）
```
触发：松开手指
时长：0.3秒
动画内容：
- 起跳动作：scaleY 0.8 → 1.1 → 1.0
- 起跳特效：脚下出现光圈和粒子
- 身体拉伸：起跳瞬间拉伸

示例代码：
{
  "name": "jump",
  "duration": 0.3,
  "keyframes": [
    { "time": 0, "scaleY": 0.8, "scaleX": 1.0 },
    { "time": 0.1, "scaleY": 1.15, "scaleX": 0.9 },
    { "time": 0.3, "scaleY": 1.0, "scaleX": 1.0 }
  ],
  "effects": ["jump_burst"]
}
```

#### 空中（Air）
```
触发：跳跃后在空中
时长：根据滞空时间
动画内容：
- 旋转：轻微自转 (0-15度)
- 摆动：手脚摆动
- 表情：紧张或期待的表情

高度影响：
- 低高度：小幅摆动
- 中高度：明显旋转和摆动
- 高高度：大幅旋转，紧张表情
```

#### 着陆（Land）

**普通着陆：**
```
时长：0.4秒
动画内容：
- 压缩：scaleY 1.0 → 0.85 → 1.0
- 缓冲：轻微弹跳

示例代码：
{
  "name": "land_normal",
  "duration": 0.4,
  "keyframes": [
    { "time": 0, "scaleY": 1.0 },
    { "time": 0.15, "scaleY": 0.85, "easing": "easeIn" },
    { "time": 0.4, "scaleY": 1.0, "easing": "spring" }
  ]
}
```

**完美着陆：**
```
时长：0.6秒
动画内容：
- 着陆压缩 + 弹跳
- 庆祝动作：举手、跳跃
- 表情：开心、眨眼
- 特效：星星、闪光

示例代码：
{
  "name": "land_perfect",
  "duration": 0.6,
  "keyframes": [
    { "time": 0, "scaleY": 1.0, "rotation": 0 },
    { "time": 0.15, "scaleY": 0.85, "easing": "easeIn" },
    { "time": 0.3, "scaleY": 1.1, "rotation": 10, "easing": "spring" },
    { "time": 0.45, "scaleY": 1.0, "rotation": -5 },
    { "time": 0.6, "scaleY": 1.0, "rotation": 0 }
  ],
  "effects": ["perfect_sparkle", "star_burst"]
}
```

#### 失败（Fail）
```
触发：掉出平台
时长：1.0秒
动画内容：
- 掉落：旋转下落
- 表情：惊讶 → 失望
- 结束：躺在地上/消失

示例代码：
{
  "name": "fail",
  "duration": 1.0,
  "keyframes": [
    { "time": 0, "y": 0, "rotation": 0 },
    { "time": 0.5, "y": 200, "rotation": 180, "easing": "easeIn" },
    { "time": 1.0, "opacity": 0 }
  ]
}
```

## 三、平台动画

### 3.1 平台基础动画

#### 生成动画
```
触发：平台进入屏幕
时长：0.3秒
动画内容：
- 缩放：0 → 1.0
- 透明度：0 → 1
- 缓动：backOut (弹入效果)

示例代码：
{
  "name": "platform_spawn",
  "duration": 0.3,
  "keyframes": [
    { "time": 0, "scale": 0, "opacity": 0 },
    { "time": 0.3, "scale": 1.0, "opacity": 1, "easing": "backOut" }
  ]
}
```

#### 消失动画
```
触发：平台离开屏幕或超时
时长：0.3秒
动画内容：
- 缩放：1.0 → 0
- 透明度：1 → 0
- 可选：碎裂、融化等特效
```

#### 完美着陆反馈
```
触发：角色完美着陆
时长：0.5秒
动画内容：
- 平台震动：轻微抖动
- 涟漪效果：从着陆点扩散
- 发光：平台边缘发光

示例代码：
{
  "name": "platform_perfect_land",
  "duration": 0.5,
  "keyframes": [
    { "time": 0, "glowIntensity": 0, "shake": 0 },
    { "time": 0.1, "glowIntensity": 1.0, "shake": 5 },
    { "time": 0.5, "glowIntensity": 0, "shake": 0 }
  ],
  "effects": ["ripple", "glow_pulse"]
}
```

### 3.2 特殊平台动画

#### 摇晃平台（如奶盖茶）
```
时长：循环
动画内容：
- 摇摆：rotation -5deg → 5deg
- 频率：2秒/周期
- 幅度：根据着陆力度调整

示例代码：
{
  "name": "platform_shake",
  "duration": 2.0,
  "loop": true,
  "keyframes": [
    { "time": 0, "rotation": -5 },
    { "time": 1.0, "rotation": 5, "easing": "easeInOut" },
    { "time": 2.0, "rotation": -5 }
  ]
}
```

#### 旋转平台（如披萨）
```
时长：循环
动画内容：
- 旋转：0deg → 360deg
- 速度：15度/秒

示例代码：
{
  "name": "platform_rotate",
  "duration": 24.0,
  "loop": true,
  "keyframes": [
    { "time": 0, "rotation": 0 },
    { "time": 24.0, "rotation": 360 }
  ]
}
```

#### 移动平台
```
动画内容：
- 平移：根据配置轨迹移动
- 缓动：根据移动类型（线性/正弦）

示例代码：
{
  "name": "platform_move_horizontal",
  "duration": 4.0,
  "loop": true,
  "keyframes": [
    { "time": 0, "x": -50 },
    { "time": 2.0, "x": 50, "easing": "easeInOut" },
    { "time": 4.0, "x": -50, "easing": "easeInOut" }
  ]
}
```

#### 开合平台（如生蚝）
```
触发：周期开合
时长：5秒/周期（3秒闭，2秒开）
动画内容：
- 闭合：scaleX 1.0
- 张开：scaleX 0.6
- 特效：张开时有粒子

示例代码：
{
  "name": "platform_open_close",
  "duration": 5.0,
  "loop": true,
  "keyframes": [
    { "time": 0, "scaleX": 1.0, "state": "closed" },
    { "time": 2.8, "scaleX": 1.0 },
    { "time": 3.0, "scaleX": 0.6, "state": "open" },
    { "time": 4.8, "scaleX": 0.6 },
    { "time": 5.0, "scaleX": 1.0 }
  ]
}
```

  "velocity": { "x": { "min": -10, "max": 10 }, "y": { "min": -50, "max": -20 } },

## 五、动画性能优化

### 5.1 优化策略

**对象池：**
```javascript
// 粒子对象池
class ParticlePool {
  constructor(prefab, size) {
    this.pool = [];
    this.active = [];
    // 预创建对象
    for (let i = 0; i < size; i++) {
      this.pool.push(this.createParticle());
    }
  }
  
  get() {
    return this.pool.length > 0 ? this.pool.pop() : this.createParticle();
  }
  
  recycle(particle) {
    particle.reset();
    this.pool.push(particle);
  }
}
```

**GPU加速：**
- 使用 transform 和 opacity 做动画
- 避免触发重排（layout）
- 使用 will-change 提示浏览器优化

**LOD（细节层次）：**
- 远距离：简化动画
- 近距离：完整动画
- 屏幕外：暂停动画

**帧率控制：**
```javascript
// 自适应帧率
if (fps < 30) {
  // 降低特效质量
  particleCount *= 0.7;
  enableSimpleEffects();
}
```

### 5.2 性能预算

| 动画类型 | 最大数量 | 复杂度 |
|---------|---------|--------|
| 角色动画 | 1 | 高 |
| 平台动画 | 10 | 中 |
| 粒子特效 | 100 | 高 |
| UI动画 | 5 | 低 |
| 背景特效 | 1 | 中 |
