# 吃了么 - 新手引导流程

## 一、引导设计原则

### 1.1 设计目标
- **降低门槛**：让零基础玩家快速上手
- **建立信心**：通过简单操作获得成就感
- **传达乐趣**：展示游戏核心乐趣点
- **不强制**：可跳过，不影响游戏体验

### 1.2 引导原则
- **循序渐进**：从简单到复杂
- **即时反馈**：操作后立即看到结果
- **容错设计**：允许失误，鼓励重试
- **趣味性**：避免枯燥说教

---

## 二、引导流程总览

```
首次启动
   ↓
开场动画（剧情引入）
   ↓
基础教学（1-1关卡内）
   ↓
核心机制教学（1-1至1-3）
   ↓
进阶技巧提示（1-4至1-10）
   ↓
完成第一章 → 解锁自由游戏
```

---

## 三、详细引导流程

### 3.1 开场动画（首次启动）

**时长**：约30秒
**触发**：首次进入游戏
**可跳过**：是（右上角跳过按钮）

**动画脚本：**
```
场景1：阿饱的卧室（5秒）
- 阿饱躺在床上，看着美食地图
- 旁白："阿饱是个热爱美食的博主"

场景2：收到神秘信件（8秒）
- 窗外飞进一封信
- 阿饱打开，是"厨神令"的传说
- 旁白："传说中，收集所有美食印记，就能获得厨神令"

场景3：下定决心（7秒）
- 阿饱起身，背上背包
- 旁白："为了成为真正的美食评论家，阿饱踏上了旅程"

场景4：第一关（10秒）
- 阿饱来到豆浆铺
- 旁白："从最简单的早餐开始，一起探索美食的世界吧！"
- 显示"开始游戏"按钮
```

**UI元素：**
- 跳过按钮：右上角，"跳过"文字，白色半透明背景
- 开始按钮：底部居中，主按钮样式
- 进度点：底部，显示当前场景

---

### 3.2 基础教学（1-1关卡 - 强制引导）

**目标**：学会蓄力跳跃的基本操作
**时长**：约1-2分钟
**强制**：是（必须完成才能继续）

#### 步骤1：认识界面（5秒）

**引导内容：**
```
高亮显示：角色（阿饱）
文字提示："这是阿饱，你的美食向导"
语音："我是阿饱，跟我一起探索美食吧！"

高亮显示：平台
文字提示："跳上这些美食平台前进"

高亮显示：蓄力条（在屏幕底部）
文字提示："按住屏幕蓄力，松开跳跃"
```

#### 步骤2：第一次蓄力（10秒）

**引导操作：**
```
文字提示："按住屏幕开始蓄力"

玩家按住后：
- 显示蓄力条增长动画
- 播放"咕嘟"蓄力音效
- 文字提示："蓄力越多，跳得越远"

高亮显示：抛物线预测线
文字提示："这条线显示跳跃轨迹"

提示松开：
文字提示："松开手指，跳起来！"
```

#### 步骤3：第一次跳跃（10秒）

**成功着陆后：**
```
播放成功音效
显示"完美！"文字
+10分分数动画

阿饱语音："太棒了！"
文字提示："成功跳到平台上！"
```

**如果失败：**
```
播放失败音效
显示"再试一次"按钮
文字提示："没关系，再试一次！"

第三次失败后：
显示提示线："按这条线蓄力试试"
降低难度：平台变大
```

#### 步骤4：连续跳跃（30秒）

**引导内容：**
```
文字提示："继续跳到下一个平台"

成功3次后：
文字提示："连续成功！保持连击！"
显示连击数：x3

高亮显示：连击数
文字提示："连击越多，分数越高"
```

#### 步骤5：完美着陆教学（15秒）

**引导内容：**
```
高亮显示：平台中心
文字提示："落在中心位置有额外奖励"

成功完美着陆：
播放"叮咚"音效
显示"完美！" + 金币特效
+30分动画

阿饱语音："完美着陆！"
文字提示："完美着陆得分更高！"
```

### 3.3 核心机制教学（1-1至1-3关卡）

#### 1-1 关卡后半段：障碍物初识

**触发时机**：第5个平台

**引导内容：**
```
高亮显示：油条障碍物
文字提示："小心！有移动的油条！"

演示动画：
- 显示障碍物移动轨迹
- 虚线指示安全时机

文字提示："等油条移开再跳"
或"快速跳过去"

成功通过后：
阿饱语音："灵巧！"
获得"躲避专家"徽章碎片
```

#### 1-2 关卡：移动平台教学

**触发时机**：第2个平台开始移动

**引导内容：**
```
文字提示："这个平台在移动！"

高亮显示：移动轨迹线
文字提示："虚线显示移动方向"

文字提示："预判位置，提前起跳"

成功着陆：
阿饱语音："时机把握得正好！"
```

#### 1-3 关卡：计时机制教学

**触发时机**：第一个计时平台

**引导内容：**
```
高亮显示：倒计时数字
文字提示："注意倒计时！"

数字变化：3→2→1
文字提示："时间到平台会消失"

文字提示："在倒计时结束前跳起来"

成功通过：
文字提示："准时！"
+10%分数奖励提示
```

### 3.4 进阶技巧提示（1-4至1-10）

**从1-4关卡开始，引导转为轻提示，不再强制**

#### 技巧提示系统

**触发方式：**
- 首次遇到新机制时
- 玩家连续失败2次时
- 检测到玩家可能有困难时

**提示类型：**

1. **蒸汽遮挡（1-4）**
```
气泡提示："蒸汽会挡住视线，记住位置再跳"
显示：记忆标记提示
```

2. **大小平台（1-5）**
```
气泡提示："小平台更难，要更精准哦"
显示：落点圈大小对比
```

3. **精准判定（1-6）**
```
气泡提示："汤包很脆弱，落在正中心！"
显示：中心点高亮
```

4. **分支选择（1-7）**
```
气泡提示："咸甜任选，都可以通关"
显示：两条路线预览
```

---

### 3.5 功能解锁引导

**随着游戏进度，逐步解锁功能：**

| 解锁时机 | 功能 | 引导内容 |
|---------|------|---------|
| 完成1-1 | 暂停功能 | "点击左上角可以暂停游戏" |
| 完成1-2 | 道具系统 | "获得道具！点击图标使用" |
| 完成1-5 | 连击系统详解 | "连击x5！分数翻倍！" |
| 完成第一章 | 第二章解锁 | "解锁午餐篇！新的挑战等着你！" |
| 获得徽章 | 徽章系统 | "获得新徽章！去徽章墙看看吧" |
| 首次失败 | 复活功能 | "观看广告可以复活继续" |
| 累计1000分 | 排行榜 | "你上榜了！查看排名" |

## 四、引导界面设计

### 4.1 引导气泡

**样式规范：**
```
位置：屏幕中央偏下
背景：#FFFFFF，90%透明度
边框：2px solid #FF9F43
圆角：20px
内边距：24px
最大宽度：80%

文字：
- 字号：24px
- 颜色：#2F3542
- 行高：1.5

箭头：
- 指向被引导元素
- 橙色 (#FF9F43)
- 8px宽，12px高
```

**动画效果：**
```
出现：
- 缩放：0.8 → 1.0
- 透明度：0 → 1
- 时长：0.3s
- 缓动：spring

消失：
- 缩放：1.0 → 0.9
- 透明度：1 → 0
- 时长：0.2s
```

### 4.2 高亮遮罩

**样式规范：**
```
遮罩层：
- 颜色：#000000
- 透明度：50%
- 全屏覆盖

高亮区域：
- 镂空显示
- 边框：3px solid #FF9F43
- 发光效果：box-shadow: 0 0 20px #FF9F43
- 脉冲动画：scale 1.0 → 1.05，循环

可点击：
- 高亮区域内可交互
- 区域外点击无效
```

### 4.3 手势提示

**按住提示：**
```
图标：手指按下图标
动画：上下浮动
文字："按住屏幕"
位置：屏幕底部中央
```

**滑动提示：**
```
图标：手指滑动图标
动画：左右滑动轨迹
文字："左右滑动选择"
位置：对应区域
```

## 六、引导数据统计

### 6.1 采集指标

| 指标 | 说明 | 用途 |
|------|------|------|
| 引导完成率 | 完成引导的玩家比例 | 评估引导有效性 |
| 步骤流失率 | 每个步骤的放弃率 | 优化引导流程 |
| 首次通关时间 | 完成1-1的时间 | 评估上手难度 |
| 引导跳过率 | 跳过引导的比例 | 评估引导吸引力 |
| 首次付费时间 | 首次消费的时间点 | 优化付费引导 |

### 6.2 A/B测试

**测试方案：**
```
方案A：强制完整引导（当前）
方案B：简化引导（只教基础）
方案C：无引导（自由探索）

对比指标：
- 次日留存率
- 7日留存率
- 首次通关率
- 付费转化率
```
