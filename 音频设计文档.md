# 吃了么 - 音频设计文档

## 一、音频整体风格

### 1.1 风格定位
- **整体风格**：轻松愉快、节奏明快、治愈系
- **音乐情绪**：积极向上，带有美食的温馨感
- **音效风格**：卡通可爱，有Q弹感
- **情绪曲线**：根据章节主题变化（早餐轻松→午餐忙碌→下午茶甜蜜→晚餐热闹→宵夜温馨）

### 1.2 音频格式规范

| 类型 | 格式 | 采样率 | 压缩格式 | 音量标准 |
|------|------|--------|---------|---------|
| BGM | MP3 | 44.1kHz | 192kbps | -12dB ~ -8dB |
| 音效 | WAV/OGG | 44.1kHz | 无损/128kbps | -6dB ~ -3dB |
| 语音 | MP3 | 44.1kHz | 128kbps | -9dB ~ -6dB |
| 环境音 | MP3 | 44.1kHz | 128kbps | -20dB ~ -15dB |

### 2.3 BGM播放规则

```
播放逻辑：
1. 音乐无缝循环播放
2. 章节切换时淡入淡出（0.5秒）
3. 暂停时音乐音量降至30%
4. 复活广告时静音
5. 支持玩家关闭BGM

音量控制：
- 主音量：0-100%
- BGM音量：0-100%
- 默认BGM音量：70%
```

## 五、音频技术规范

### 5.1 音频管理器架构

```
AudioManager
├── BGMPlayer (背景音乐播放器)
│   ├── 当前音乐
│   ├── 淡入淡出控制
│   └── 音量控制
├── SFXPlayer (音效播放器)
│   ├── 音效池管理
│   ├── 3D音效支持
│   └── 音量控制
├── VoicePlayer (语音播放器)
│   ├── 语音队列
│   └── 语音优先级
└── Mixer (混音器)
    ├── Master (主音量)
    ├── BGM (音乐音量)
    ├── SFX (音效音量)
    └── Voice (语音音量)
```

### 5.2 音效池配置

| 音效类型 | 池大小 | 预加载 | 释放策略 |
|---------|--------|--------|---------|
| 跳跃音效 | 5 | 是 | 关卡结束 |
| 金币音效 | 8 | 是 | 即时释放 |
| UI音效 | 3 | 是 | 即时释放 |
| 环境音效 | 2 | 否 | 动态加载 |
| 特殊音效 | 1 | 否 | 按需加载 |

### 5.3 音频优化

**内存优化：**
- BGM：仅加载当前章节音乐
- 音效：常用音效预加载，其他动态加载
- 语音：按需加载，播放后释放

**性能优化：**
- 限制同时播放的音效数量（最大8个）
- 距离衰减（3D音效）
- 音频压缩（移动端适配）

**格式优化：**
- iOS：使用AAC格式
- Android：使用Vorbis格式
- 通用：MP3作为备用
