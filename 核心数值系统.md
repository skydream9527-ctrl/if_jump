# 吃了么 - 核心数值系统设计文档

## 一、物理系统参数

### 1.1 角色物理属性

| 参数名称 | 数值 | 说明 |
|---------|------|------|
| 重力加速度 | 9.8 m/s² | 标准重力 |
| 角色质量 | 1.0 | 标准化质量 |
| 最大水平速度 | 8.0 单位/秒 | 蓄力满时的水平速度 |
| 起跳角度 | 45° | 固定角度，优化跳跃距离 |
| 角色碰撞体积 | 0.8×0.8 单位 | 方形碰撞框 |

### 1.2 蓄力系统

| 参数名称 | 数值 | 说明 |
|---------|------|------|
| 最大蓄力时间 | 1.5 秒 | 蓄力达到最大值的时间 |
| 蓄力速度 | 66.7%/秒 | 每秒增加蓄力百分比 |
| 最小蓄力比例 | 20% | 低于此比例不跳跃 |
| 蓄力到距离系数 | 2.33 单位/100% | 100%蓄力=3.5单位距离 |
| 蓄力进度条分段 | 25%/50%/75%/100% | 视觉提示刻度 |

**蓄力计算公式：**
```
蓄力百分比 = 按住时间 / 1.5秒 × 100%
跳跃距离 = 蓄力百分比 × 0.0233 (单位)
跳跃速度 = 蓄力百分比 × 0.08 (单位/秒)
```

### 1.3 跳跃物理

| 参数名称 | 数值 | 说明 |
|---------|------|------|
| 起跳初速度公式 | v = √2gR | R为跳跃距离，g为重力 |
| 跳跃高度公式 | h = R/4 | 抛物线最高点高度 |
| 滞空时间公式 | t = √(2h/g) × 2 | 总滞空时间 |
| 空气阻力 | 0.95 | 速度衰减系数 |

**跳跃轨迹计算示例（100%蓄力，3.5单位距离）：**
```
起跳水平速度: 8.0 单位/秒
跳跃高度: 3.5/4 = 0.875 单位
起跳垂直速度: √(2×9.8×0.875) ≈ 4.15 单位/秒
滞空时间: 2×0.875/4.15 ≈ 0.42 秒
```

## 二、评分系统

|---------|------|----------|
| 基础着陆分 | +10分 | 成功跳到平台上 |
| 中心奖励 | +30分 | 落在平台中心区域 |
| 完美着陆 | +50分 | 落在中心点（误差<0.1单位） |

### 2.2 连击系统

| 连击数 | 分数倍率 | 特效 |
|--------|---------|------|
| 1-2 | ×1.0 | 无 |
| 3-5 | ×1.2 | 显示"Good!" |
| 6-9 | ×1.5 | 显示"Great!" |
| 10-14 | ×2.0 | 显示"Excellent!" |
| 15+ | ×3.0 | 显示"Perfect!"+金色特效 |

**连击规则：**
- 连续完美着陆（中心区域）算连击
- 普通着陆连击+1，但不计入倍率
- 掉落或失败连击清零
- 连击中断时显示最终连击数和奖励

### 2.3 特殊加分

## 六、成就与徽章

### 6.1 徽章等级

| 徽章类型 | 颜色 | 难度 | 奖励 |
|---------|------|------|------|
| 铜徽章 | 棕色 | 简单 | 50金币 |
| 银徽章 | 银色 | 普通 | 100金币 |
| 金徽章 | 金色 | 困难 | 200金币 |
| 钻石徽章 | 彩色 | 极限 | 500金币+专属皮肤 |

### 6.2 星级评定标准

| 星级 | 条件 | 奖励倍率 |
|------|------|---------|
| ⭐ | 通关即可 | ×1.0 |
| ⭐⭐ | 达到目标分数 | ×1.5 |
| ⭐⭐⭐ | 完美通关（无失误+高分） | ×2.0 |

**目标分数计算公式：**
```
目标分数 = 基础分 × (1 + 章节系数 × 关卡序号)
基础分 = 1000
章节系数 = 0.2

例：第二章第3关 = 1000 × (1 + 0.2×2×3) = 2200分
```

## 九、数值平衡检查

### 9.1 经济系统平衡

```
收入预估（每日）：
- 每日任务：150金币
- 签到：60金币（平均）
- 游戏获得：200金币（假设20分钟游戏时间）
- 总计：约410金币/天

支出预估：
- 复活消耗：50金币/次
- 道具购买：30金币/个
- 皮肤解锁：1000-5000金币

平衡性：玩家需要约3-7天解锁普通皮肤，2-3周解锁稀有皮肤
```
